## 已实现的排序算法

### 分区
- 左枢轴快速排序
- 中枢轴快速排序
- 中值枢轴快速排序

### 合并
- 合并排序

### 选择
- 选择排序
- 堆排序

### 插入
- 插入排序
- 壳排序
- 树排序

### 交换
- 冒泡排序
- 侏儒排序
- 鸡尾酒调酒器排序
- 交换排序
- 奇偶排序
- 梳状排序

### 非比较
- 统一键桶排序
- 最低有效数字 (LSD) 基数排序
- 最高有效数字 (MSD) 基数排序

### 其他
- 煎饼排序
- 双音排序
- 臭皮匠排序
- 慢排序
- Bogo 排序

### 系统
- JavaScript 排序

## 开发信息

这个项目是用 [Angular CLI](https://github.com/angular/angular-cli) 版本 12.2.10 生成的。

使用 [维基百科摘要 API](https://en.wikipedia.org/api/rest_v1/#/) 获取排序算法摘要和文章链接。
